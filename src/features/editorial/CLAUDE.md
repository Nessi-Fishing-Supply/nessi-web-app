# Editorial Feature

## Overview
Discovery and editorial content — shop highlights, maker stories, featured listings, and social proof.

## Components
- `shop-highlight/` — Shop hero card with preview grid
- `maker-story-block/` — Pull quote + narrative editorial block
- `featured-listing-card/` — 16:9 hero listing card
- `species-browse-row/` — Horizontal scroll species selector
- `social-proof-strip/` — Stats strip or activity feed
- `price-drop-alert/` — Price drop banner or saved-tab row
- `recently-sold-ticker/` — Auto-scrolling transaction feed

## Future
- `services/editorial.ts` — Editorial content data
