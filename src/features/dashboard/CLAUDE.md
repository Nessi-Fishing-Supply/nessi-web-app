# Dashboard Feature

## Overview
Seller dashboard with KPIs, quick actions, and performance metrics.

## Components
- `kpi-stat-tile/` — Label + value + trend indicator
- `quick-action-card/` — Icon + label + badge + chevron action card
- `sparkline/` — SVG micro-chart (no library)
- `shop-upgrade-prompt/` — Conversion card for shop upgrade

## Future
- `services/analytics.ts` — Dashboard analytics data
- `hooks/use-dashboard.ts` — Tanstack Query hooks
